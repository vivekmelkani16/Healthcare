# 🏥 CarePulse - Healthcare Patient Management System

CarePulse is a modern healthcare patient management application that allows patients to register, book appointments, and enables administrators to manage appointments efficiently.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Healthcare.git
```

Go into the project folder

```bash
cd Healthcare
```

Install dependencies

```bash
npm install
```

Create a `.env.local` file and add your environment variables.

Run the project

```bash
npm run dev
```

Open

```
http://localhost:3000
```

---

---

## 📦 Build

```bash
npm run build
```

Run production

```bash
npm start
```
